Info


This site is dedicated to my much missed mum and dad, Marion and Ken Nicholls, and is a way for family members and friends
to add photos, group them into memories and share them with one another so we can all remember them both with love and
affection.


Logging In

In order to view and add photos firstly you will need to Sign In. To do this you will need your email address and password that
should have been provided to you.

Press the Sign In menu option.



Enter the email address and password provided. Once you are successfully logged in you will stay
logged in on this browser on this device until you explicitly Log Out.

 

Change Password




Once logged on it is best to change the password provided to you with one known only to yourself.

To do this press click on your name in the top right hand corner and press the Change Password menu
option. Enter your existing and new password. Your new password must contain at least one lowercase and one
uppercase letter and be at lease 6 characters in length.


Edit Personal Details




Once you have changed your password you can update your personal details. For example
add a profile picture and correct date of birth.

To do this press click on your name in the top right hand corner and press the Edit menu
option.


View Photos

Once you have sccessfully logged in you will have access to the Photos menu option.




To view existing photos press the Photos List menu option. This will show you
all of the photos currently available, split in to pages of 6 photos, in alphabetical order.
The pagination options at the bottom will allow you to scroll through each page in order
or jump to a specific page.



Clicking the title of a photo will allow you to view or edit the photo depending
on whether you added the photo originally or are an admin user. You will also be able view
and add comments relating to this photo. Clicking directly on a photo will show thatphot in
full screen. Click on the photo to return to normal size.



Also, depending on whether you originally added the photo or are an admin user, there
will be a Delete option. This will permanently delete this photo so beware!



If there are any comments associated with the photo there will be an option to view these comments.



Next to the title there is an option to view a Slide Show. This will show a pop-up which will allow you to scroll
through all of the photos using the arrows. Click on the main photo to view this full screen. If a search
is active then only those photos restricted by the search will be shown in the slide show.



 

Add a Photo.




To add a new photo press the Add Photo menu option under the Photos menu.
You will need to add a unique title in order to identify the photo and browse for a
photo to upload. The rest of the fields are optional. You can associate tags and people
with the photo. To select an existing tag click the tag dropdown and select the tag. To add a
new tag to this list press the Add Tag button and enter the new tag. If you want this tag to be
automatically associated with this photo make sure the Auto Select Tag checkbox is checked.



It is a similar process to associate people with this photo.

The date of the photo is the date that the photo was taken. If this is not known leave blank.

If you only know the year select Jan 1st of that year. If you only know the decade then
select Jan 1st on the first year of that decade.





There are 2 methods of selecting the photo to be uploaded. File System allows you to select
directly from the file system of the device you are using. A tablet or mobile device will also allow you
to take a photo - which is handy for your profile picture. Web allows you to choose photos directly from the
web or from a number of social media sites which you are a member of. Please use this option sparingly
as I am using the free service and the bandwidth is limited and there are a limited number of uploads a
month and a finite number in total.

 

Chat Messaging

Presing the Chat menu option will bring up the Chat screen. This will allow you to send
a message to all other logged in users. The message will then appear in the Messages area.
There is also an option to Send Location. This will send your current location to all other logged
in users in the form of a link which will then take them to your location highlighted on google maps.

You can also send a message to a single user only. To do this click the user you want to send a
message to under Active Users and this will show the above screen but the message will only be
sent to the selected user.


Memories

Memories work in a similar way to Photos the only difference being is that you can associate zero or more
existing photos with a memory. Each memory, once addeed, will have the option of viewing all of the
associated photos in a slide show.




Searching





The list of memories and photos that are seen in the list option can be restricted by pressing the
Search button when in the List option. This will allow you to restict the number of photos or memories
to those which match your search criteria.

How the entered search criteria restrict the memories or photos depends on the Match Criteria:



Case Sensitive

If selected then the search will only return photos which match the case of the search criteria. The
default is not to restict on case.

Match All

If this is checked then all entered critria must match. For example if you try and match a photo
with a tag of 'beach' and a person of 'dad' then only photos where both are present will be returned
(the default). If this is unchecked then photos where either are present will be returned.

Match Criteria:

Starts With

This will match any field which starts with the entered value. For example a photo with the title 'dad by the beach'
will be matched with a search title of 'dad'.

Exact

This will match any field which exactly matches the entered value. For example a photo with the title 'dad by the beach'
will be matched with a search title of 'dad by the beach' A search title of 'dad' would not match.

Contains

This will match any field which contains the entered value. For example a photo with the title 'dad by the beach'
will be matched with a search title of 'dad by the beach', 'dad', 'beach', or 'by the'.

Note that all of the above are amalgamated together when working out if a photo matches a search. For example,
in the above example, 'Dad' will only match if Case Sensitive search was turned off.

To change the match criteria click Show Match Criteria  in the search dialogue.

Note that any field left blank will not be used when matching and hence it is not possible to search
for, say, memories with no description.
